How to use this site:



Welcome to NYC Schools, a web application to help New York City residents find information about secondary schools in the city.

Each icon on the map represents a school, whose respective information page can be visited by clicking the icon and it's link.

To the right, you can search and filter for individual schools. Again, clicking on the school name will display the school's information page.
