About This Application

This application was built to help parents of prospective New York City high school students find the ideal school for their children based on location, academics, and school values.

This project was made possible using:

SQL to query data, stored on Google Firebase

AngularJS to display data for individual schools, and configure site routing

HTML5 and CSS for general site development and design
